## Prompter Telegram Bot (aiogram 3.x + Gemini + Groq)

## Quickstart

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
```

Create `.env`:

- `BOT_TOKEN=...`
- `GEMINI_API_KEY=...`
- `GROK_API_KEY=...` (опционально, для Groq.com и fallback)

Run:

```bash
python main.py
```
